## WHAT WE BUILT

Three AI features integrated into the music recommender:

1. **Confidence Scoring** - Each recommendation now includes a confidence %
   - Shows how certain the system is (0-100%)
   - Based on: how many criteria matched (genre, mood, energy, acoustic)
   - 0 matches → 0% confident
   - 4 matches → 100% confident

2. **User Feedback Learning** - System learns from what users actually like
   - Records: "I showed you a song, did you like it?"
   - Analyzes: "Out of 20 recommendations, user liked 16"
   - Learns: "80% accuracy, I'm doing well"

3. **Weight Adaptation** - System improves its scoring weights over time
   - If accuracy > 50%: increase weights (be more confident)
   - If accuracy < 50%: decrease weights (be more cautious)
   - Weights actually change based on feedback

---

## HOW IT FITS INTO THE SYSTEM

### Original System (Rule-Based)
```
Song CSV → Recommender (hardcoded weights) → Recommendations → Output
                35 genre + 40 mood + 40 energy + 10 acoustic
```

### Enhanced System (AI-Powered)
```
Song CSV → Recommender (LEARNABLE weights) → Recommendations → Output
           ↓                    ↑
           └─ User Feedback ────┘
              (likes/dislikes)
                  ↓
           WeightLearner
           (adjusts weights based on accuracy)
```

The AI doesn't replace the scoring algorithm—it IMPROVES it by learning
which weights work best for each user.

---

## THE FLOW: End-to-End

Here's exactly how a user gets a recommendation and the system learns:

STEP 1: User wants recommendations
   Input: genre="pop", mood="happy", energy=0.8, acoustic=False

STEP 2: Recommender scores all songs
   For each song, check:
   ✓ Does genre match? +35 points
   ✓ Does mood match? +40 points
   ✓ Is energy close? +0-40 points (uses LEARNABLE weights)
   ✓ Is acousticness right? +10 points
   
   COMPUTE CONFIDENCE:
   If 3 out of 4 criteria match → 75% confident
   If 2 out of 4 criteria match → 50% confident

STEP 3: Return top-5 with confidence
   [
     (Sunrise City, 125 points, 100% confident),
     (Gym Hero, 85 points, 75% confident),
     (Rooftop Lights, 80 points, 50% confident),
     ...
   ]

STEP 4: User reacts
   "I liked Sunrise City" ✓
   "I didn't like Rooftop Lights" ✗

STEP 5: Record feedback
   FeedbackTracker stores:
   - Song: Sunrise City
   - User: pop/happy/0.8/electronic
   - Prediction: 100% confident
   - Actual: liked ✓

STEP 6: Accumulate more feedback (repeat steps 2-5 many times)
   After 20 recommendations:
   - Liked: 16
   - Disliked: 4
   - Accuracy: 80%

STEP 7: System learns and adapts
   WeightLearner says:
   "Accuracy is 80%, which is > 50%, so weights are good!
    Let me increase them slightly to be more confident"
   
   New weights:
   old: genre=35, mood=40, energy=40, acoustic=10
   new: genre=35.7, mood=41.1, energy=41.1, acoustic=10.3 (1.3× adjustment)

STEP 8: Recommender gets new weights
   Next time I score a song:
   ✓ Genre match: +35.7 (instead of 35) ← slightly higher
   ✓ Mood match: +41.1 (instead of 40) ← slightly higher
   
   Same songs score slightly higher overall → more confident recommendations

STEP 9: Cycle continues
   More recommendations → more feedback → another learning round
   Weights keep adapting = system keeps improving
